About This Course

This class primarily serves as an introduction to Python and, more broadly, to programming. Our goal is to work towards certification in Information Technology Specialist – Python (ITS-303).

Since we meet only once a week, I highly encourage you to dedicate as much time as possible to practicing Python and gaining hands-on experience. I also strongly recommend starting small personal projects as soon as possible. Don't worry about whether they seem achievable at first—what matters most is that they are meaningful or interesting to you.

ITS Specialist

The ITS Specialist exam primarily consists of multiple-choice questions and similar variations. These variations often include:

  • Drag-and-drop components, where you align descriptions with the correct terms.
  • Code completion tasks, requiring you to arrange the correct syntax within a program.
  • Drop-down selections, where you choose the appropriate value to complete a statement.
  • Additionally, you may be required to read a block of code and predict its output to demonstrate your understanding.
The exam will be administered through Certiport. Therefore, you will need to create an account with Certiport, which can be done using the link provided below. You can also review the outlined learning objectives.

002 - Data Types

Data Types

In this set of slides primarily covers data types in Python, with a particular focus on the concept of mutability. While this may seem theoretical at first, understanding mutability provides valuable context for how different data types behave and how their methods function. To explore these concepts, we used functions such as is and id to demonstrate specific behaviors. We also discussed different ways to declare integer values using various base systems, such as binary and hexadecimal. Additionally, we introduced:
  • Numerics & collections
  • Slicing and indexing
  • Type conversion between different data types
  • None Object Data Type
  • Comparison Operators
  • Comparing and contrasting strings
  • in keyword (Built-in function)
  • Arithmetic Operators
  • Character code values and order of operations
The provided exercise focuses on code tracing and practising Python code execution. I'd like you to carefully read the given code segment and then figure out its output. It may be helpful to take notes explaining why you expect a particular outcome. Once you have made your prediction, I recommend typing the code into an IDE of your choice and running it. Be sure to investigate any unexpected behavior and experiment with modifications to confirm your understanding of the logic.
