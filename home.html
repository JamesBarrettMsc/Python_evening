About This Course

This class primarily serves as an introduction to Python and, more broadly, to programming. Our goal is to work towards certification in Information Technology Specialist – Python (ITS-303). Since we meet only once a week, I highly encourage you to dedicate as much time as possible to practising Python and gaining hands-on experience. I also strongly recommend starting small personal projects as soon as possible. Don't worry about whether they seem achievable at first—what matters most is that they are meaningful or interesting to you.

The ITS Specialist exam primarily consists of multiple-choice questions and similar variations. These variations often include:

  • Drag-and-drop components, where you align descriptions with the correct terms.
  • Code completion tasks, requiring you to arrange the correct syntax within a program.
  • Drop-down selections, where you choose the appropriate value to complete a statement.
  • Additionally, you may be required to read a block of code and predict its output to demonstrate your understanding.
The exam will be administered through Certiport. Therefore, you will need to create an account with Certiport, which can be done using the link provided below. You can also review the outlined learning objectives.

004 - Operators

This section explores operators in Python, which are symbols used to perform operations on variables and values. Arithmetic operators handle mathematical calculations, including floor division, which returns the largest whole number quotient. Comparison or relational operators are used to evaluate relationships between values, including string comparisons. Logical operator s help in making decisions by combining Boolean expressions, while their behavior with non-Boolean types is also examined. Additionally, bitwise operators manipulate individual bits within integers, enabling low-level data processing.

Topics covered are the following:

  • What is an Operator?
  • Arithmetic operators
  • Floor Division
  • Comparison/Relational operators
  • Comparing strings
  • Logical operators
  • Logical operators for Non-Boolean types
  • Bitwise operators

007- Flow Control (I)

This section covers fundamental control flow structures in Python. Sequential execution follows a step-by-step approach, while selection structures, such as if-else, if-elif-else, and nested if statements, enable decision-making based on conditions. Repetition is handled using loops, including the while loop, which continues execution as long as a condition is met. Additionally, the use of else with loops is explored, demonstrating how it executes when a loop completes without interruption.

Questions related to the following topics:

  • Sequential
  • Selection
  • Repetition

  • nested if
  • While loop
